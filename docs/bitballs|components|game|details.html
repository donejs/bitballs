<game-details>

  • module
bitballs/components/game/details

{Module}

 
  • source

Provides a custom element that allows a user to either view a game or edit a game's stats.

Module

To create a <game-details> element pass the Session and a Game id like:

<game-details
    {session}="session"
    {game-id}="gameId"
    ></games-details>
